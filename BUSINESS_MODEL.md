# KeyHub Central — Business Model & Monetization Plan

**Version:** 1.0
**Last Updated:** January 2, 2026
**Status:** Planning

---

## Executive Summary

KeyHub Central operates as a multi-revenue platform with three monetization layers:

1. **Internal Contractor Tools** — Freemium SaaS for KTS contractors
2. **Lead Subscriptions** — B2B lead generation for external contractors (Keynote Digital)
3. **White-Label Licensing** — Platform-as-a-service for other businesses

---

## 4. Recruitment Integration

### Overview

Built-in recruiting module to source contractors from job boards and manage the hiring pipeline.

### Job Board Integrations

| Platform | Integration Type | Cost | Timeline |
|----------|------------------|------|----------|
| **Indeed** | Job Sync API (GraphQL) | Free API calls | 6 weeks |
| **ZipRecruiter** | Partner API (REST) | Free API calls | 4 weeks |
| **LinkedIn** | Jobs Posting API | Paid per post | 4 weeks |

## 5. Compliance Integration

### Background Checks

**Recommended Provider:** Checkr

| Package | Price | Includes |
|---------|-------|----------|
| Basic | ~$35 | SSN trace, national criminal |
| Standard | ~$50 | + County criminal, sex offender |
| Pro | ~$85 | + MVR, employment verification |

**Integration Flow:**

```
Applicant Status = "Offer Accepted"
            │
            ▼
┌───────────────────────────────────┐
│  KeyHub calls Checkr API          │
│  POST /candidates + /invitations  │
└─────────────────┬─────────────────┘
                  │
                  ▼
┌───────────────────────────────────┐
│  Candidate receives email         │
│  Completes Checkr flow            │
│  (SSN, consent, identity)         │
└─────────────────┬─────────────────┘
                  │
                  ▼
┌───────────────────────────────────┐
│  Checkr webhook → KeyHub          │
│  Status: clear | consider         │
└─────────────────┬─────────────────┘
                  │
        ┌─────────┴─────────┐
        ▼                   ▼
     Clear              Consider
        │                   │
        ▼                   ▼
  Auto-advance         Admin review
   to W-9 step          required
```

### W-9 Collection

**Recommended Provider:** TaxBandits

| Service | Price |
|---------|-------|
| W-9 collection + e-signature | ~$2-3/form |
| TIN matching (SSN/EIN verification) | ~$1/match |
| 1099 generation (year-end) | ~$3-5/form |

**Integration Flow:**

```
Background Check = "Clear"
            │
            ▼
┌───────────────────────────────────┐
│  KeyHub calls TaxBandits API      │
│  POST /w9/request                 │
└─────────────────┬─────────────────┘
                  │
                  ▼
┌───────────────────────────────────┐
│  Contractor receives email/link   │
│  Fills W-9 form + e-signs         │
└─────────────────┬─────────────────┘
                  │
                  ▼
┌───────────────────────────────────┐
│  TaxBandits runs TIN Match        │
│  Validates SSN/EIN with IRS       │
└─────────────────┬─────────────────┘
                  │
                  ▼
┌───────────────────────────────────┐
│  Webhook → KeyHub                 │
│  Store PDF in Firebase Storage    │
│  Update contractor compliance     │
└─────────────────┬─────────────────┘
                  │
                  ▼
      Proceed to insurance upload
```
